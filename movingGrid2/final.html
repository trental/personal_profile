"paperclip" is a trading platform that lets the user make the decision about the value of their belongings.

List items to trade and items wanted, and the application automatically connects an unlimited number of users for multi-person trades.
User connection is created with a recursive function - it searches for people that want what you have.

It does this by traversing the database as if it were a directed graph.
"paperclip" is a vision born out of a collaborative effort. I (Trent) created and coded the connection algorithm along with the rest of the back-end.

The "paperclip" tech stack includes:
- React
- NodeJS
- MongoDB
- Mongoose
- Express
paperclip features full CRUD on user items and user needs.
